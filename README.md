# 🐶 Maze Solver – Pathfinding Visualizer

An interactive maze-solving game that visualizes pathfinding algorithms with animations, sound effects, and a leaderboard.

## 🚀 Features
- BFS, DFS, and A* algorithms
- Real-time visualization with animations
- Moving agent (puppy 🐶)
- Maze generator (DFS-based)
- Sound effects + background music
- Leaderboard (stores best times)
- Interactive grid (draw walls)

## 🧠 Algorithms Explained

### BFS (Breadth First Search)
- Guarantees shortest path
- Explores level by level

### DFS (Depth First Search)
- Explores deeply first
- Not optimal (may give longer path)

### A* (A Star)
- Uses heuristic (Manhattan distance)
- Fast and optimal
- Best overall

## 🎮 How to Use
1. Draw walls or generate maze
2. Select algorithm (BFS / DFS / A*)
3. Watch the puppy solve the maze 🐶
4. Check stats and leaderboard
